Less obvious perks of using 'live' data resources include logins and payments. How could Twitter be used in these two scenarios?
Companies realize that social media can help promote their products by reaching a large audience without spending a penny. To this end, being able to quickly access a users information makes API resources invaluable.
Being able to seamlessly transfer user info through the various API’s out there gives giant advantages when drawing in and retaining users who just want a simple interface.
Login-By a user allowing their login information to be recorded, it allows companies
to track the users movements, likes, dis-likes and can make for a unique marketing tool when spreading a companies message. Companies can allow the same user login info to their site and make it simpler and easier for users to move along.
Payment-By storing users “payment type” data, companies make it easier for a user to
simply say yes to previously stored payment-type information to purchase goods and services. Eliminating the hassle of filling out billing information and credit card numbers every time increases the chance of a user purchasing a product or service because it’s just easier for them.
Visit Programmableweb.com and search for JSON-P data sources. Why would we need JSON-P, specifically?
When domains communicate with each other they follow the ‘Same Origin’ policy which prohibits their interaction between each other. This is mostly for the purpose of security. Not allowing one domain affect another.
The usefulness of JSON-P in this instance allows a payload of JSON data to be transferred from one API to another.
The “P”, which stands for padding, is function that is “wrapped” around the data. This allows the self contained function the ability to parse the data using javascript. Sending the information with a function allows any other API the ability to render ANY JSON data because the function needed to display the data is already included.
JSON-P analogy JSON =a new un-assembled bike with assembly directions written in Chinese.
-P=is a Chinese speaking mechanic that’s included with the bike and assembles it for you so you don’t need to figure out how to read the directions.
