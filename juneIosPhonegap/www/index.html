Visit Programmableweb.com and search for JSON-P data sources. Why would we need JSON-P, specifically?
When domains communicate with each other they follow the ‘Same Origin’ policy which prohibits their interaction between each other. This is mostly for the purpose of security. Not allowing one domain affect another.
The usefulness of JSON-P in this instance allows a payload of JSON data to be transferred from one API to another.
The “P”, which stands for padding, is function that is “wrapped” around the data. This allows the self contained function the ability to parse the data using javascript. Sending the information with a function allows any other API the ability to render ANY JSON data because the function needed to display the data is already included.
JSON-P analogy JSON =a new un-assembled bike with assembly directions written in Chinese.
-P=is a Chinese speaking mechanic that’s included with the bike and assembles it for you so you don’t need to figure out how to read the directions.

Mobile Apps: Please list two apps (either Android or iOS, or both) that are made specifically to support Accessibility. AT & T currently has a media kit containing two apps that assist with accessibility. It focuses mainly on the visually impaired. The two cross-platform applications are called Mobile Speak and Mobile Magnifier. Mobile Speak-Is a basic screen reader app that guides the user through the interface using audio cues. Mobile Magnifier-This application allows the user to adjust their screen in a variety of ways including magnification and the adjusting of the color spectrum for better accessibility by the visually impaired Devices: Please list two devices (and some of their features) that are made specifically to support Accessibility. Freedom Scientific has developed a braille reader for the blind that interacts with your computer through Bluetooth. It’s an excellent example of innovation in accessibility for the visually impaired. NCH software has a product called Debut. It is screen capture software designed to capture video from a variety of source and allows that video to have things such as captions added to the videos. You can also add time stamps, change the color and burn directly to a dvd if need be. I can see how this would help in areas such as education and training where the ability to make these adjustments would not only benefit the visually impaired, but also allow users to convey thoughts and ideas in a way that plain text never could.
